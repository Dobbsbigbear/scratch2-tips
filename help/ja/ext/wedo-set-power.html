モーターまたはライトの電源を指定の値に設定し、電源を入れます。 値の範囲は0〜100です(0はパワー無し、100は最大パワーです。)。

上下の矢印キーを使ってモーターの速度を変更します。


ドロップダウンメニューからLEGO-WeDo対応のデバイスを選択することができます。:
  • If you have only one motor, use the "motor" option, which will set the motor''s power regardless of where it is connected on the WeDo hub.
  • 2つのモーターを別々に制御したい場合は、"motor A"と"motor B"を別々に使用することができます。
  • "lights" sets the lights'' power regardless of where they are connected on the WeDo hub.
  • "everything"は、接続されているすべてのモーターとライトの電源を設定します。

～のパワーを～にするを使うには、LEGO WeDoハブ経由でコンピュータに接続されたWeDoモーターまたはライトが必要です。
