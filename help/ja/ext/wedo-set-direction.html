モーターの向きを設定します。 すでにオフの場合は、このブロックはオンにしません。

毎秒、回転する向きを変えながら、10秒間モーターを動かします。


最初のドロップダウンメニューからWeDoモーターを選択可能です。:
  • If you have only one motor, use the "motor" option, which will set the motor''s direction regardless of where it is connected on the WeDo hub.
  • 2つのモーターを別々に制御したい場合は、"motor A"と"motor B"を別々に使用することができます。
  • "all motors"は、接続されているすべてのモーターの方向を設定します。

2番目のドロップダウンメニューから方向を選択できます。

set directionを使うには、LEGO WeDoハブ経由でコンピュータにWeDoのモーターを接続する必要があります。
